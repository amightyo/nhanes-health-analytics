# Raw Data (NHANES)

This folder contains the original, unmodified NHANES SAS Transport (.XPT) files.
These files are **never edited**, only read by Python scripts.

Files included:
- P_DEMO.XPT  (Demographics + weights + design variables)
- P_BMX.XPT   (Body Measures, includes BMI)
- P_PAQ.XPT   (Physical Activity Questionnaire)
- P_SLQ.XPT   (Sleep Questionnaire)
